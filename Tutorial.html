Set up

Setting up LadderBot is easy.

The step below will walk you though setting up your first ladder and some basic of running a tournament.

3 Steps

Step 1 Add Bot

Add the bot to your server and accept the server permissions LadderBot needs to run.

Add To Discord


Step 2 Create New Tournament

When you are ready to set up a tournament, first make sure you have the ladder_organizer role that was created when LadderBot joined.

Next run the /new_ladder command (see all commands here) with: the team size limit (1 for 1v1s, 2 for 2v2s...), the max amount of teams that can sign up, and the max distance two teams can be in rank to challenge (usually 1, 2 or 3).

Next fill out the info in the modal window and hit submit! Ladder bot will go on to make some tournament channels for players to sign up, see team rankings and captain and mod activities.

Step #3 Open Registrations

When you are read for teams to start joining, hit the open registrations button in the moderators channel. This will allow team to sign up and start challenging each other! For more info use /help or join our server
